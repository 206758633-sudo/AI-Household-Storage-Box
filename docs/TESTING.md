# 测试说明

## 当前覆盖

测试入口：

```bash
cd mini_program
npm test
```

覆盖范围：

- 本地归档路由：记账、倒数、资产、打卡。
- 倒数规则：过期生日滚到下一年。
- 生肖与星座：1963 年 5 月 23 日为属兔、双子。
- 资产日均：有购买日期时输出日均。
- 状态筛选：14 天内订阅命中临近。
- 月报统计：金额与打卡次数聚合。
- 快速归档：`create_entry` 使用本地规则先返回；AI 复核通过 `refine_entry` 后台触发。
- 开发者工具兜底：云环境未配置或云函数超时时，小程序端使用本地测试存储继续验证主流程。
- 编辑链路：详情页点击编辑后可修改主要字段并保存，资产可补购买日期。
